# Shopping Cart Application

JavaFX shopping cart app with database localization.

## Requirements

- Java 21
- Maven
- MariaDB

## Database Setup

1. Open MariaDB and run:

```
schema.sql
```

## Run the App

```
mvn javafx:run
```

## Usage

1. Select a language from the dropdown
2. Enter number of items
3. Enter price and quantity for each item
4. Click Calculate — results are saved to the database
